## 3) Canvas

The canvas shows shapes of only current annotation tool:

![Preview](/gif/current.gif)


You can modify the shapes you draw on the canvas:

![Preview](/gif/modify2.gif)

You can either accept or ignore suggested shape:

![Preview](/gif/sugF.gif)


## 4) Labels

When a shape is drawn on the canvas, an input pops out in the labels menu,
You can either add it , or delete it:

![Preview](/gif/label.gif)

Each category shows the number of saved labels ,you can see the saved Labels by clicking on the category:

![Preview](/gif/see.gif)


When hovering over the shapes on the canvas, the label connected to the shape is highlighted in the labels menu:

![Preview](/gif/hover.gif)
